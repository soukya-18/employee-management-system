#A web-based Employee Management System built using **Flask**, **Python**, and **MySQL**.
.

## Features
- Role-based login (Admin, HR, Manager, Employee)
- Add, edit, delete employees
- Upload employee photos
- Salary analytics and dashboard
- Export employee data to Excel
- Secure authentication with password hashing

## Tech Stack
- Python (Flask)
- MySQL
- HTML, CSS
- Matplotlib
- OpenPyXL
